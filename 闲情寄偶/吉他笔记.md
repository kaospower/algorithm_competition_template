# 扫弦  
1.手型:大拇指和食指交叉,交点大致在大拇指指腹处  
2.下扫动作:扫弦初始位置在音孔上边缘处,自然扫到琴的挡板处,扫弦时要同时摆动手腕和小臂,同时注意手腕在下扫过程中不要外翻  
         初学时扫一到四弦,一定要扫到一弦,一弦扫不到是因为手腕外翻  
3.上扫动作:大拇指指甲侧卡在一弦处,同时与琴弦成一定角度,不要把大拇指全靠在弦上,然后做类似抛硬币的动作向上扫弦,初学时扫一到三弦  
         上扫时,小臂,手腕,手指都要摆动  
扫弦一定要连贯,不要中途停顿,那样会变成琶音,扫弦可以从琶音开始练,比如下扫时,从六弦开始向下,之后起始位置逐步提高,转变成扫弦  
下扫与上扫的结合,下扫到最低处时,拇指与食指松开,拇指向上移动至卡在一弦处,然后向上扫弦,熟练后这将变成一个连贯的动作  
4.切音:下扫至一二弦时,转动手腕使声音停止,熟练后会变成擦擦的声音,注意切音结束时拇指要尽量卡在一弦处,方便后续上扫  
# 和弦转换  
两个要点:保留和先后  
两个和弦都要按的位置称为保留指,和弦转换时,利用保留指,可以减少需要移动的手指,从而加快转换速度  
如果两个和弦没有保留指,一个和弦结束后,就把手指全部松开  
当从一个和弦转换到另一个和弦时,可以发现有些弦是不需要按的,这样就先松开该弦手指,然后弹响,之后考虑后面的弦  
如果该弦在和弦中需要按住,再移动手指  
由于大多数节奏是从上到下拨弦,因此按和弦时尽量掌握从上到下依次按弦的动作,这对和弦转换非常重要  
核心思想是还不需要弹的弦可以后按,马上要弹的弦必须先按
