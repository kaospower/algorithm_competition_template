# 1.切比雪夫距离

## 3648.覆盖网格的最少传感器数目

数学
$对于一个传感器,其可以覆盖范围为边长为size=2*k+1的正方形$
$答案即为最少用几个正方形可以覆盖全部区域,因此用\lceil\frac{n}{size}\rceil.\lceil\frac{m}{size}\rceil即可$

```python
#3648(https://leetcode.cn/problems/minimum-sensors-to-cover-grid/s)
class Solution:
    def minSensors(self, n: int, m: int, k: int) -> int:
        return (n+2*k)//(2*k+1)*((m+2*k)//(2*k+1))
```

# 2.绝对值化简

## 3649.完美对的数目

经过绝对值化简转化为|a|<=|b|<=2|a|
按照绝对值大小排序,此时数组中的数的绝对值具有单调性,可以用双指针解决

```python
#3649(https://leetcode.cn/problems/number-of-perfect-pairs/)
class Solution:
    def perfectPairs(self, nums: List[int]) -> int:
        nums.sort(key=abs)
        ans=l=0
        for i,b in enumerate(nums):
            #如果|b|>2|a|,左指针需要右移,直到|b|<=2|a|位置
            while abs(b)>abs(nums[l])*2:
                l+=1
            #总计有i-l个数对
            ans+=i-l
        return ans
```
