# 1.Abel求和公式

$\displaystyle\sum_{i=m}^n a_ib_i=a_nS_n-\displaystyle\sum_{i=m}^{n-1}(a_{i+1}-a_i)S_i,S_i=\displaystyle\sum_{k=m}^i b_k$

3500题用到了这个公式

# 2.阶乘尾0个数:勒让德定理

n!的尾0个数,取决于n!可以分解出多少个10,10=2x5,需要知道n!中质因子2的个数,5的个数,两者较小值,即为n!的尾0个数

定义n!的质因子分解中的质数p的指数为$\nu_p(n!),有\nu_p(n!)=\displaystyle\sum_{i=1}^{\infty}\lfloor\frac{n}{p^i}\rfloor$

172题,793题用到了这个公式

# 3.串联数组的元素和问题

定义$arr^k表示k个arr串联得到的数组,s是数组arr的元素和$

$定理:如果s>0,那么arr^2的最大子数组和一定横跨两个arr,不会在其中一个arr中间$

# 4.从n个数中选奇数个数的方案数

从n个不同元素中,选奇数个数,有$2^{n-1}种选法$

1863题,3558题用到了这个结论

# 5.每次删除两个不同元素,最多能删多少次

操作次数上界为$min(\lfloor\frac{n}{2}\rfloor,n-m)$

变形题:每次操作删除至多两个不同元素,删除所有元素,最少要操作多少次?

结果为$max(\lceil\frac{n}{2}\rceil,m)$
