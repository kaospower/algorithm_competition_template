**3394 判断网格图能否被切割成块**

分两种情况,扫描横坐标和扫描纵坐标,这样矩形就可以看成投影到x,y轴的线段
将每个线段看成两个事件,左端点看成进,状态是1,右端点看成出,状态是0
将所有事件加入数组,排序
遍历所有事件,遇到1就+1,遇到0就-1,如果当前状态清0,说明是一个合法分割线
最后统计合法分割线条数是否>=3(因为最后边界也算1条)
注意进和出的状态不能反过来,因为遇到一个线段的尾和另一个线段头重合时,应该先结算出后结算进
因此出必须排在前面,所以出用状态0,进用状态1

注意此题也可以转化成合并区间问题求解,见区间贪心章节

```python
#3394(https://leetcode.cn/problems/check-if-grid-can-be-cut-into-sections/?envType=problem-list-v2&envId=NYf6Nvxg)
class Solution:
    def checkValidCuts(self, n: int, rectangles: List[List[int]]) -> bool:
        ax,ay=[],[]
        for x1,y1,x2,y2 in rectangles:
            ax.append((x1,1))
            ax.append((x2,0))
            ay.append((y1,1))
            ay.append((y2,0))
        ax.sort()
        ay.sort()
        def f(a):
            s=cnt=0
            for i,(x,y) in enumerate(a):
                s+=1 if y else -1
                if s==0:
                    cnt+=1
            return cnt>=3
        return f(ax) or f(ay)
```
