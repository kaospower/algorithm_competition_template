**3027.人员站位的方案数II**

此题思路比较特殊

枚举右,维护左
将坐标按照横坐标从小到大排序,如果横坐标相同,则按照纵坐标从大到小排序
两两枚举点对,如果第二个点的纵坐标<=第一个点的纵坐标并且>纵坐标最大值
更新最大值,同时答案+1

也可以考虑离散化+二维前缀和的解法

```python
#3027(https://leetcode.cn/problems/find-the-number-of-ways-to-place-people-ii/?envType=problem-list-v2&envId=hgtpjpHr)
class Solution:
    def numberOfPairs(self, points: List[List[int]]) -> int:
        points.sort(key=lambda p:(p[0],-p[1]))
        ans=mx=0
        for i,(x1,y1) in enumerate(points):
            mx=-inf
            for x2,y2 in points[i+1:]:
                if mx<y2<=y1:
                    mx=y2
                    ans+=1
        return ans
```
