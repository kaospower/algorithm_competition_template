# 5.合并区间

**3394.判断网格图能否被切割成块**
将有重叠的区间合并,看最终得到的区间数是否>=3,这就转化成了56题

本题也可以用扫描线解决,参见扫描线章节

```python
#3394(https://leetcode.cn/problems/check-if-grid-can-be-cut-into-sections/?envType=problem-list-v2&envId=NYf6Nvxg)
class Solution:
    def checkValidCuts(self, n: int, rectangles: List[List[int]]) -> bool:
        def f(a):
            #按照左端点排序
            a.sort()
            pre=ans=0
            for l,r in a:
                #左端点>=之前右端点,新区间
                if l>=pre:
                    ans+=1
                #无论是否合并,pre都更新为pre和r的较大值
                pre=max(pre,r)
            return ans>=3
        ax=[(x1,x2) for x1,y1,x2,y2 in rectangles]
        ay=[(y1,y2) for x1,y1,x2,y2 in rectangles]
        return f(ax) or f(ay)
```
