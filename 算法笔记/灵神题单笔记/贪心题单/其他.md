**3002.移除后集合的最多元素数**

本题本质是求集合间的关系,是数学类型的贪心

第一种思考方式:

从s1,s2独有部分挑元素,且不能超过n//2,如果还不够,就从交集挑,且总和不能超过n

```python
#3002(https://leetcode.cn/problems/maximum-size-of-a-set-after-removals/?envType=problem-list-v2&envId=NYf6Nvxg)
min=lambda x,y:x if x<y else y
class Solution:
    def maximumSetSize(self, nums1: List[int], nums2: List[int]) -> int:
        n=len(nums1)
        s1,s2=set(nums1),set(nums2)
        common=len(s1&s2)
        return min(min(n//2,len(s1)-common)+min(n//2,len(s2)-common)+common,n)
```

第二种思考方式:

从s1中选至多n//2,s2中选至多n//2,如果两部分有交集,就返回s1和s2的并集,否则返回两者的和

实际实现时两者取min即可

```python
min=lambda x,y:x if x<y else y
class Solution:
    def maximumSetSize(self, nums1: List[int], nums2: List[int]) -> int:
        n=len(nums1)
        s1,s2=set(nums1),set(nums2)
        return min(min(n//2,len(s1))+min(n//2,len(s2)),len(s1|s2))
```
