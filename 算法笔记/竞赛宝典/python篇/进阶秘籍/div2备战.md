参加 Codeforces Div2 比赛，常见题目难度从 A（入门）到 E（较难），需要掌握的算法与数据结构可以按以下几个层次来准备：

------

## 一、基础篇（刷 A、B 题必备）

1.  **数组与字符串操作**
    -   数组/字符串的遍历、分块、滑动窗口
    -   前缀和／差分数组（Prefix Sum / Difference Array）
2.  **排序与查找**
    -   标准排序（快速排序、归并排序）
    -   二分搜索（Binary Search）及其变体
3.  **贪心思想**
    -   区间调度、活动选择
    -   最小化／最大化策略构造
4.  **图论基础**
    -   图的表示（邻接表/邻接矩阵）
    -   DFS / BFS 遍历
    -   简单连通性、最短路（Unweighted BFS）
5.  **集合与哈希**
    -   STL 的 `set`、`unordered_set`、`map`、`unordered_map`
    -   去重、计数、滑动窗口中的哈希技巧

------

## 二、中级篇（刷 C、D 题常见）

1.  **数据结构**
    -   **堆与优先队列**：解决中位数维护、最小／最大合并等
    -   **并查集（DSU）**：连通性、动态连通问题
    -   **线段树 / 树状数组（Fenwick Tree）**：区间查询与区间修改
    -   **Sparse Table**：静态 RMQ（最小/最大/ gcd）
2.  **动态规划（DP）**
    -   线性 DP （背包、最长上升子序列）
    -   滚动数组优化
    -   区间 DP 的入门（括号匹配、区间合并）
3.  **二分+判定**
    -   在答案空间上二分（“二分答案”）
    -   检查函数设计：可行性、贪心判定
4.  **数学与数论**
    -   快速幂、模逆元、费马小定理
    -   素数筛（埃拉托斯特尼筛法）、质因数分解
    -   组合数预处理（阶乘和逆元表）
5.  **字符串算法**
    -   KMP（前缀函数）、Z 函数
    -   Rolling Hash（双 hash 防碰撞）
    -   字典树（Trie）

------

## 三、高级篇（刷 Div2 D、E 题、乃至更高难度）

1.  **高级数据结构**
    -   **可持久化线段树 / 分块（主席树）**
    -   **Treap / FHQ Treap**：平衡二叉搜索树
    -   **平衡树（Splay / 红黑树）**：STL 中 `std::set` 底层
    -   链剖 / 树链剖分（HLD）
2.  **网络流 / 最小费用流**
    -   Dinic、ISAP、SAP
    -   SPFA + 流量增广、费用流
3.  **几何算法**
    -   旋转卡壳（Convex Hull）、点线关系
    -   最近点对、半平面交
4.  **数论进阶**
    -   中国剩余定理、杜教筛、莫比乌斯函数
    -   原根与离散对数
5.  **高级 DP 技巧**
    -   状压 DP
    -   树形 DP
    -   插头 DP

------

## 四、备战策略

1.  **循序渐进**：先刷 Div2 A/B，再过渡到 C/D。
2.  **专题练习**：每学一种算法或 DS，就在 Codeforces 上找对应标签（Tag）题目刷。
3.  **阅读优秀题解**：比赛后及时做题解总结，累积模板代码。
4.  **模拟赛与计时**：按比赛环境计时做题，练习快速读题和代码实现能力。

------

掌握以上算法与数据结构，并结合大量实战训练，相信你能在 Codeforces Div2 中取得好成绩！祝刷题顺利。
