# 1.数据清洗和预处理  
# 数据清洗  
数据清洗是指发现并纠正数据文件中可识别的错误 ，包括检查数据一致性、处理无效值和缺失值等。  
数据清洗目的:提高数据质量，保证数据分析、数据挖掘等后续工作的准确性和可靠性。  

数据去重,去噪,格式化,深度清洗,语义去歧,事实性核验,价值观与合规对齐  

# 数据预处理  
数据集成方法,缺失值识别与分类,异常值检测与处理,数据标准化方法

# 数据降维技术

# 2.数据增强  
数据增强是深度学习和机器学习中一种常用的技术，旨在通过对原始数据进行各种变换或扩充，  
生成新的训练样本，从而提升模型的泛化能力、防止过拟合，并增强对噪声和变形的鲁棒性。 

计算机视觉,自然语言处理,大语言模型,音频,视频,多模态  

# 3.数据标注  
语音标注:语言识别,语音分割,情感分析  
视频标注:行为识别,动作识别,目标跟踪  
文本标注:分词,词性标注,命名实体标注  
图像标注:目标检测,图像分类,语义分割
