# 知识型智能体
适用场景:
企业知识管理  
智能客服与支持  
装备运维助手  
QHSE问答  

核心操作流程:  
1.知识库构建  
2.知识库维护  
3.智能体创建与关联  
4.配置与联动发布  

搭建流程:  
进入智能体平台  
1.我的空间-知识库-创建知识库  
2.填写知识库描述信息  
(1)名称和介绍,按照实际业务情况填写  
(2)向量模型,默认选择1024  
(3)大模型:选择该知识库对应的模型  
(4)全员公开回答:若知识库有一定权限访问要求,建议选择"否"  
3.上传文档  
在点击上传文档之前可创建不同文档对应的分类,提前做好文档整理工作  
上传成功后,可以在文档界面看到该文件,选择拆分方式  
(1)智能拆分:  
智能解析:主要针对文本格式,如word,txt等  
深度OCR:主要针对以图片形式展示的文档(如:PDF,图片等),或需要识别图片中的文字部分  
(2)自定义拆分:如果文档内有规整的目录格式或分隔符,可选择按目录拆分或按分隔符拆分  

FAQ(Frequently Asked Questions)  
先下载模版,再点击上传  
一般用于智能客服  
4.知识切片查看  
点击"查看"可查阅该文档被切分的知识,左侧是原文可进行比对  
若大模型对文档的分割有问题,可通过右上角的7个功能按钮进行手工调整:  
(1)上方插入知识块  
(2)下方插入知识块  
(3)向下合并  
(4)向上合并  
(5)添加问法  
(6)编辑  
(7)删除  
5.问答体验  
点击"问答体验"可提前感受基于该文档提出的问题,体验回答准确情况,右侧展示命中的原文  
右上角可切换大模型  
检索设置,问答设置都可以修改  
topk:大模型生成答案的规则,从k个候选值中随机选择一个作为答案  
温度值:生成随机性  
6.文件权限管理  
...-文档问答权限
点击添加角色标签,选择之前已创建好的角色(此处以审计角色为例).则为审计角色的用户有权限对该文档进行问答和编辑  
7.版本启用  
文档管理-版本管理-发布版本-启用   
8.知识库发布(可选)  
点击发布  
9.知识库分享(可选)  
分享-添加分享-选择账号姓名-编辑权限-选择权限-确定  
10.智能体创建  
创建智能体-知识型智能体  
11.智能体配置  
(1)名称和标签,按照业务实际情况或待匹配知识库填写  
(2)描述和指令,点击右上角星星按钮,大模型自动生成  
(3)知识库选择,选择刚才创建的知识库  
(4)右上角选择对应的大模型  
12.版本启用  
配置管理-版本管理  
点击"发布"按钮,"启用"按钮才会出现  
13.智能体发布  
点击发布  
可勾选智能体商店,API市场,网页分享等  
14.智能体应用配置  
平台管理-应用基础配置-勾选需启动的智能体-配置调用智能体的相关信息-提交  
15.审核管理  
刚提交的应用需在此处进行审核  

平台管理-应用管理-新建应用,可供后端开发人员调用API  

# 任务型智能体  
适用场景:  
差旅要素提取  
钻台危险行为识别  
文章仿写  
智能问表  

核心操作流程:  
1.创建任务链  
2.配置组件  
3.创建任务链版本  
4.创建智能体  
5.关联发布  

1.任务链创建  
我的空间-任务链-创建任务链  
填写任务链基本信息  
2.任务链编辑  
组件库可供选择  
具备对文档内容进行解析输出报告的功能  
模版建议:  
(1)定义AI角色,明确目标  
(2)指定任务要求  
(3)指定期望的输出格式和内容  
(4)设置输出标准  
3.任务链调试  
4.任务链保存并启用  
保存-配置管理-版本管理-发布-启用  
5.任务链发布  
6.智能体创建  
我的空间-智能体-任务型智能体  
7.智能体配置  
(1)填写智能体基础信息  
(2)选择任务链  
8.智能体启用  
保存-配置管理-版本管理-发布-启用  
9.智能体发布  
10.智能体应用配置  
平台管理-应用基础配置-勾选需启用的智能体-配置调用智能体的相关信息-提交  
11.审核管理
