安装步骤

1.打开anaconda

2.conda create - name torch-gpu python=3.10

3.conda activate torch-gpu启动pytorch环境

4.conda install pytorch torchvision torchaudio pytorch-cuda=11.8 -c pytorch -c nvidia(见官网)

5.完成
